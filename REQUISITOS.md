# Requisitos - birutinha

## Runtime
- **Node.js** (sem versão específica no package.json)

## Stack
- React 18 + TypeScript
- Vite 4
- Tailwind CSS + PostCSS + Autoprefixer

## Dependências principais
- `@supabase/supabase-js` — Supabase
- `@vimeo/player` — Player de vídeo Vimeo
- `three` — 3D (Three.js)
- `gsap` — Animações
- `axios` — HTTP client
- `lucide-react` — Ícones
- `react-router-dom` — Roteamento

## Variáveis de ambiente (.env.example)
- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

## Como rodar
```bash
npm install
npm run dev
```
